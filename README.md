## 技術スタック
- typescript
- React
- firebase

## 機能
-Email + Password 認証
-Password リセット機能 (リセット用Email配信)
-ユーザープロフィール登録
-Google認証¥
-ログイン情報のRedux ToolKitによるグローバル状態管理
-Cloud firestoreによるデータベース(投稿データとコメントデータ)
-Cloud storageによる画像保存 (Tweet画像とAvatar画像)
